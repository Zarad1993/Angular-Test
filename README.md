# Angular-Test
Angular Test reply for WalletHub

# Steps to Run the applicaiton

## Clone repo
```sh
git clone https://github.com/Zarad1993/Angular-Test.git
```

## On Command Line
```sh
bower install
```

## Run Server
```sh
python -m SimpleHTTPServer
```

## Open browser
```sh
http://localhost:8000/#/
```
